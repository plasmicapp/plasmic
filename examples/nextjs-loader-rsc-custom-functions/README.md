A [Next.js](https://nextjs.org/) **App Router** project using the Plasmic
[`@plasmicapp/loader-nextjs`](https://www.npmjs.com/package/@plasmicapp/loader-nextjs)
loader with **React Server Components**, showing how to use **custom functions** in
queries that are prefetched on the server.

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000). To use your own project,
replace the `id` and `token` in [`plasmic-init.ts`](./plasmic-init.ts).
